Search Results Viewer

This view enables the user to see what schemas matched the search keyword. The keyword is displayed in the top left corner of this pane for reference purposes. Below this, a full listing of all matched schemas is displayed with the matched elements shown and the matched words highlighted. Any of these schemas can be clicked on to launch the schema viewer. Jumping to this view allows more intensive analysis to be performed on the matched schemas.

Functionality has also been included to take search results and mark them with a tag. The button to do so can be found in the top right corner of the pane. When clicked, a "Create Tag" dialog is launched where the user can assign a name to the tag as well as edit which schemas should truly be categorized with this tag.
