Importer Formats

Spreadsheet Importer

This importer is designed to import a simplified schema from a spreadsheet. The format for this file is a .xls file. Each page in the spreadsheet is declared as an entity. The columns in the first row of the spreadsheet are identified as attributes. And the domain of each attribute is then derived from the types of the data found in each of these columns. If multiple types are found in a single column, the domain defaults to string.

Domain Importer

This importer is designed to import a simple set of domains and their domain values. The format for this file is a .xls file containing three columns. The first column stores the domain name, the second column stores the domain value, and the third column represents the associated description. For columns where no domain value is given, the description is assigned to the domain. Otherwise, the description is assigned to the domain value. An example is shown below.
